# Fundamental Concept of Programming Languages

### Course Learning Objective:
-   Become a better programmer in any programming languages
-   Learn the ideas in which programming languages were built
-   The different ideas being used when programming
-   How the ideas are expressed in different languages

### What Course is about:
-   Learn basic concept of all programming languages
-   Lots of focus on Functional programming
    -   Not using mutation(assignment statements)
    -   Using *first class functions*
